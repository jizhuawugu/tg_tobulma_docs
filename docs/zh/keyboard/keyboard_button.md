# 按钮详情
![img.png](/keyboard/button_detail.png)
## 行
::: tip
表示一行显示的按钮数量。
:::

## ➕的作用

::: tip
每行按钮数量 +1。
:::

## ➖的作用

::: tip
每行按钮数量 -1。
:::

## 🔄 重置按钮

::: tip
清空所有已设置的按钮
:::

## 🔘 新增按钮 (编辑点击键盘响应内容按钮)

- 点击 **"🔘 按钮管理"** 进入设置界面。
- <img src="/keyboard/keyboard_button.png" width="250" height="250"/>
- 输入按钮名称和链接，格式如下：
```md
点击我跳转::https://t.me/ToBulmaLivebot
```
