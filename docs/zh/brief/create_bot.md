# 创建 Telegram 机器人

1. **打开 Telegram 应用**，搜索并打开 [@BotFather](https://t.me/BotFather)。

2. **开始对话**：发送 `/start` 命令。

3. **创建新机器人**：输入 `/newbot`，然后按照提示依次输入：
    - **机器人名称**：可以是任意名称。
    - **用户名**：必须以“bot”结尾（例如 `mynewbot`）。

4. **获取 API 令牌**：创建成功后，@BotFather 会生成一个独特的 API 令牌。**复制并保存**该令牌，后续需要用它来访问机器人。
