## 1. 选择用户

在“用户管理”界面，您可以从列表中选择要管理的用户。用户列表显示了多个用户的名称，如“cat”，“李华”，“·托尼”，“m”，“Roni
Sk”等。若要查找特定用户，可以使用**搜索**功能。

## 2. 排序和筛选

- **排序|新用户**：可以按不同的排序方式显示用户，或者筛选出新注册的用户，方便进行管理。
- **搜索**：输入用户名称，快速找到目标用户。
- **重置**：点击此按钮将重置筛选条件，返回默认显示。

## 专业版专享：

- **专业版用户**可查看**更多用户信息**，实现更高效的管理，提升工作体验。
